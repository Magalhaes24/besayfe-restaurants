How Vocabulary Learning Works (BPE)

This system learns vocabulary similar to Byte Pair Encoding (BPE):

  1. Observe tokens: Every ingredient, allergen, and origin is tracked in the vocabulary
  2. Record co-occurrences: When tokens appear together (e.g., "shrimp" + "shellfish allergen"), it's recorded
  3. Learn merges: Frequently co-occurring tokens merge into compounds (e.g., "white fish" → "whitefish")
  4. Apply vocabulary: All future normalization uses this learned vocabulary for better tokenization
  5. Persistent storage: Vocabulary grows across all corrections and never resets
